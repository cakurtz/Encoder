# Encoder

[Description]
There is only one java file for this assignment. Each step of the process is broken up into its own method. It starts in main, then goes to a separate method to extract the frequencies of each symbol. It then goes to the Huffman Coding methods that create the encodings of the symbols. I found my encoding at https://rosettacode.org/wiki/Huffman_coding#Java. I only had to make changes to it for the double encoding and it was just to modify it from working with chars to strings. I also added data structures to store the encodings for easy access during the decoding process. It then encodes the symbols by line in a new file in a method called encodeSingle and then decodes that file to a separate file in decodeSingle. It then calculates the entropy in a separate method and calculates the percentage difference. To compile my program, use "javac *.java". To run the program, use "java Encoder frequenciesFile k", where k is a positive integer.

[Finish]
I finished the entire project, but did not finish any extra credit attempts. I am not entirely sure if I did the percentage difference correctly in terms of what the assignment was asking for. I used the formula (efficiency - entropy) / entropy * 100.
